Basic Gradient Descent

Gradient descent is an important algorithm for minimizing a function. Having an algorithm to minimize a function is quite powerful: you can maximize a function fff by minimizing −f-f−f and you can solve a system of equations f1=k1,f2=k2,⋯fr=krf_1=k_1, f_2=k_2, \cdots f_r=k_rf1​=k1​,f2​=k2​,⋯fr​=kr​ by minimizing (f1−k1)2+(f2−k2)2+⋯(fr−kr)2(f_1-k_1)^2+(f_2-k_2)^2+\cdots (f_r-k_r)^2(f1​−k1​)2+(f2​−k2​)2+⋯(fr​−kr​)2.
